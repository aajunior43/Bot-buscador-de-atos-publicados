Como usar

📊 CSV (Excel/Planilhas)

Ideal para análise em Excel, Google Sheets ou LibreOffice. Separado por ponto-e-vírgula (;), codificação UTF-8.

Campos: id, edição, data, página, categoria, órgão, tipo, número, data do ato, assunto, valor, resumo IA.

🔧 JSON (API)

Ideal para integrações com outros sistemas, scripts Python ou dashboards. Também disponível em tempo real via /api/publicacoes.

🔗 URL direta

Você pode acessar diretamente:

/api/exportar?format=csv&apenas_inaja=true /api/exportar?format=json&tipo=Decreto
